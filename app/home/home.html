Personal Projects


I like to write code that makes people's life less buggy


Personal Bookstore

A web application that provides a platform to sell/buy books for students that are in the same school. This project initially targetted only Caldwell University students as it requires a Gmail login.

Technologies: Mongodb, Expressjs, Angularjs, Nodejs, Passportjs, Bootstrap



Random Restaurant Generator

A basic web application which uses google map api and yelp api to search nearby restaurants nearby (radius of 10 mile), then randomly select one restaurant. The search provides data such as pictures, rating, reviews, location, direction.

Technologies: AngularJS, Node.js, Google Map API, Yelp API, Bootstrap



Online Address Book

Super simple boilerplate project that uses Angularjs, Nodejs, and MongoDB as an exercise to practice CRUD cycle with MEAN Stack.

Technologies: AngularJS,Node.js, MongoDB
